# EcoDescarte

Projeto acadêmico desenvolvido para a disciplina de Fábrica de Software.

## Descrição

O EcoDescarte é um sistema voltado ao cidadão que tem como objetivo auxiliar no descarte correto de resíduos urbanos. A aplicação permite consultar informações sobre locais adequados de descarte, solicitar coleta de materiais de grande porte e registrar denúncias de descarte irregular.

O sistema busca facilitar o acesso a informações ambientais importantes e contribuir para a redução do descarte inadequado de resíduos.

## Funcionalidades

- Consultar orientações sobre descarte de diferentes tipos de materiais
- Solicitar coleta de materiais de grande porte
- Registrar denúncias de descarte irregular
- Enviar descrição e imagem em denúncias
- Receber confirmação após envio de solicitações ou denúncias

## Tecnologias utilizadas

- CSS

- Git e GitHub (versionamento do projeto)
